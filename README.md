# npm workspaces monorepo Vite + Lit + React + Storybook

demonstration of a monorepository with the following technologies:

- npm workspaces
- Vite
- Lit
- React
- Storybook
- Typescript
- Storybook

## Packages structure description

- [packages/](./packages)
  - [app/](./packages/app) - frontend application example
  - [lit/](./packages/lit) - package with lit elements
  - [react/](./packages/react) - package with react components
  - [styles/](./packages/styles) - package with CSS modules
